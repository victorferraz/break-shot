sizing options

AUTO SIZING or CUSTOM SIZE

Auto sizing

Auto Size property is the ability to read ,understand and generate screenshots from all existing breakpoints in your project. It’s also meant to read the height of an element since you set it's ID as breakshot. In that case, you’ll be given a file with limited height for each breakpoint. It’s very useful for generating responsive banners and ads. Leave it selected and let the application work for you.

Custom size

Set width and height values in order to generate custom and specific pixel dimensions. You may also set a variety of custom sizes for the same project by clicking the “+” button after each size input.
